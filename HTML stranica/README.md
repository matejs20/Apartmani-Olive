# Apartmani Olive

Statička web stranica za iznajmljivanje apartmana – projekt za kolegij Uvod u Web tehnologije.

## Opis projekta

Web prikazuje **Apartman Olive**, jednosoban apartman u Bibinjama (37m², 2+1 osoba). Stranica sadrži naslovnicu, detaljni opis s opremom i tablicom, galeriju slika i kontakt formu.

## Stranice

- `index.html` – Naslovnica s uvodom i istaknutim sadržajima
- `about.html` – Detaljne informacije, tablica s podacima, popis opreme
- `gallery.html` – Galerija slika apartmana i dvorišta
- `kontakt.html` – Kontakt podaci i HTML forma s validacijom

## Kako pokrenuti

1. Preuzmi sve datoteke u isti folder
2. Otvori `index.html` u web pregledniku
3. Ili koristi Live Server u VS Code-u

## Korištene tehnologije

- HTML5 (semantički elementi)
- CSS3 (Flexbox, Grid, CSS varijable, responzivni dizajn)
- JavaScript (samo za hamburger izbornik)
